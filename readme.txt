Projet réalisé en L1 avec Pygame avec Roche Jules, Salfary Alban, Aubry Joshua et Vallée Mathieu

C'est un jeu de type match-3 (comme Candy Crush) avec un aspect RPG :
Les gemmes rouges attaquent l'ennemi,
les bleues offrent de la défense,
les vertes soignent,
les jaunes permettent de monter une jauge qui permet de faire des gros dégats une fois celle-ci remplie.

 
Pour lancer le projet, il faut lancer 'Combat.py'.
